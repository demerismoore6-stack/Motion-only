# Motion Only

**Build Your Career. Not Just Your Audience.**

Motion Only is the all-in-one business operating system for independent artists and entertainment professionals. Manage music, contracts, finances, bookings, and team collaboration—all in one premium platform.

## 🎯 What is Motion Only?

Motion Only is NOT a social media platform. It replaces scattered spreadsheets, cloud drives, calendars, and contract tools with a single premium application designed specifically for:

- Artists & Singers
- Producers & Songwriters
- Managers & Booking Agents
- Videographers & Photographers
- Record Labels & Entertainment Companies

## 🚀 Core Features

### Modules
- **Dashboard** - Customizable business overview with key metrics
- **Music Hub** - Complete music catalog management
- **Release Planner** - Workspace for every release
- **Motion Contracts** - AI-powered legal document management
- **Deal Rooms** - Collaboration spaces for projects
- **Money Center** - Financial tracking and reporting
- **Booking Center** - Performance and event management
- **Team Management** - Invite and manage collaborators
- **Motion Vault** - Encrypted cloud storage
- **Opportunity Center** - Discover growth opportunities
- **Content Planner** - Plan creative work
- **Career Score** - Dynamic business readiness evaluation

### Design Language
- Premium, dark-mode-first interface
- Matte black backgrounds with white typography
- Motion Only yellow accent color (#FFD700)
- Glassmorphism and smooth animations
- Native iOS & Android feel

## 🛠 Tech Stack

### Frontend
- **Web**: Next.js 14 + TypeScript + TailwindCSS
- **Mobile**: React Native + Expo
- **UI**: Custom premium component library
- **State**: TanStack Query + Zustand

### Backend
- **Runtime**: Node.js 18+
- **Framework**: Express.js
- **Database**: PostgreSQL 15
- **ORM**: Prisma
- **Auth**: JWT + Passport.js
- **Storage**: AWS S3
- **AI**: OpenAI API

### DevOps
- **Containerization**: Docker
- **CI/CD**: GitHub Actions
- **Monitoring**: Sentry

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env.local

# Start development
npm run dev

# Start Docker services
docker-compose up -d
```
